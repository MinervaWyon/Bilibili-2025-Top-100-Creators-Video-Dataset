**项目名称**：Bilibili 2025 Top 100 Creators Video Dataset / B站2025百大UP主视频信息数据集

## 字段说明

| 字段名 | 类型 | 说明 |
|--------|------|------|
| BV号 | 字符串 | 视频的唯一BV号（B站内部标识） |
| 标题 | 字符串 | 视频标题 |
| 链接 | 字符串 | 视频完整URL链接 |

## 可能的分析方向

- **UP主影响力分析**：对比不同UP主的平均播放量、互动率（点赞/投币/收藏/评论等）。
- **视频内容趋势**：分析不同分区视频的受欢迎程度，热门话题或关键词提取。
- **发布规律**：探索视频发布时间与播放量的关系，UP主更新频率等。
- **互动指标相关性**：研究点赞、投币、收藏、分享之间的关联，以及与播放量的关系。
- **长尾视频挖掘**：找出播放量虽低但互动率高的“小众精品”。
